# 📌 Overview

Employee Management System is a professional web application that simplifies employee administration within an organization.

The project includes:

- Employee Management
- Department Management
- User Management
- Role-Based Access Control
- Authentication
- Reports
- Dashboard Analytics
- Activity Logs
- Excel Import & Export
- PDF Export

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/VenkataNarayanaKankata/EmployeeManagementSystem.git
```

---

## Open Project

```bash
cd EmployeeManagementSystem
```

---

## Restore Packages

```bash
dotnet restore
```

---

## Apply Database Migration

```bash
dotnet ef database update
```

---

## Run Project

```bash
dotnet run
```
